# PulsarAds — Regras do Projeto

## 🎨 Design System (NUNCA MUDAR)
- Tema: Dark space theme
- Cores: Violet (#7c3aed) + Cyan (#38bdf8)
- Fundo: #0a0a0f, #13131a
- Bordas: rgba(255,255,255,0.08)
- Tipografia: Inter (body) + Space Grotesk (display)
- Raio: 12px cards, 10px botões

## 💰 Regra Máxima
TODA implementação deve ser 100% GRATUITA.
- ❌ Nunca usar APIs pagas ou SaaS com trial
- ✅ Preferir open-source do GitHub, APIs públicas (Meta Ad Library, Pollinations, Tesseract)
- ✅ Serviços com plano free generoso (Upstash Redis, Netlify)

## 🏗️ Arquitetura
- Frontend: HTML + JS vanilla + CSS puro (sem React/Vue)
- Backend: Node.js + Express
- Extensão: Manifest V3
- Sempre entregar arquivos COMPLETOS, nunca snippets parciais

## 🤖 Modo de trabalho
- Sempre explique o que vai fazer antes de aplicar
- Peça confirmação antes de deletar arquivos
- Mantenha compatibilidade com o estilo visual atual
- Responda sempre em português do Brasil
